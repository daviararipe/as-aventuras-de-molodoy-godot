# As Aventuras de Molodoy

Jogo 2D desenvolvido em Godot Engine como projeto autoral para prática de lógica de programação, organização de código e desenvolvimento de sistemas de gameplay.

## Tecnologias
- Godot Engine
- GDScript

## Funcionalidades
- Sistema de movimentação do personagem
- Sistema de colisões e interação com o cenário
- Controle de estados do jogador
- Lógica de jogo

## Como executar
1. Abra o projeto no Godot Engine
2. Clique em "Play" para iniciar o jogo

## Objetivo do Projeto
Aplicar conceitos de programação, raciocínio lógico e boas práticas de desenvolvimento de software em um projeto prático e interativo.
